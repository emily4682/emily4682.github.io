Chemistry Reaction Simulator

Language/s Used:

C# (and Unity)

Description:

This was my computer science A-Level Project. I built this project in Unity and C#, learning new languages and software made this project challenging, especially as I had no previous coding expereince before my A-Level studies. This project was graded A*.

The project allows users to simulate chemistry reaction mechanisms, as this is a difficult part of chemistry A-Level studies. Visualising the physical mechanisms can significantly help students with their studies. Users can also find out more information about the chemicals involved in the reaction process.
